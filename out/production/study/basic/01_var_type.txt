# 변수

* variable
* 알파벳, 숫자, _, $ 로 구성
* 대소문자를 구분, 이름사이 공백X

## byte
## short : c/c++ 호환
## int : 4바이트단위의 자료형, 32바이트 초과시 long으로 처리
* 기본적으로 java에서 정수는 4바이트로 저장
* long : 숫자 마지막에 식별자 L 사용 => 8바이트로 저장하도록
## char : 문자형 2byte, 문자형에서 숫자로 encoding, 숫자에서 문자형 decoding

## float(4바이트), double(8바이트) : 실수 자료형
* float를 사용하는 경우 : 숫자 마지막에 식별자 F 사용

## boolean : 논리형
